OUR PROJECT

The purpose of this project was to build a NEATO that autonomously navigate through a course simulating a real road, with road signs, stop lights, lanes, intersections, and obstacles.

The Neato navigates on a 6x7 grid of 18 inch squares, with red tape marking lanes and indicating which squares are not navigable. All streets can be navigated in both direction, but are only wide enough for one Neato, with the exception of a two-lane section at the bottom that allows for passing. Intersections are indicated by AprilTags sign along the road, and road signs are indicated by signs of varying shapes. Obstacles are represented by boxes.

To the right is a video of the Neato navigating through a portion of the track, combined with a sensor view that shows the lanes and signs the Neato is identifying.

COMPONENTS

PATH PLANNING

The robot calculates the shortest path between points, then generates instructions on how to get there.

SIGN RECOGNITION

The robot can detect and identify different signs shapes, and perform different behaviors at each sign.

LANE FOLLOWING

The robot uses edge detection algorithms to detect lanes and make adjustments to stay within the track boundaries.

OBSTACLE AVOIDANCE

When the robot gets close to an obstacle, it can detect the proximity and change lanes to avoid it.
